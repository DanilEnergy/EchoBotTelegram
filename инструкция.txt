1. Версия Python: 3.10.11
2. pip install -r requirements.txt | Установка зависимостей.
3. В папке config находится файл config.py, в двойных кавычках вставьте свой токен бота Telegram.
4. Запустить файл main.py

В папке handlers хранятся методы для взаимодействия пользователя с ботом.
В папке texts находится файл text.py, в нём хранятся все текста. Исключение: константы из config.py
В файле main.py регистрация хендлеров.
